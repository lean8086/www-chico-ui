Class Detail

ch.Layer

Is a contextual floated UI-Object.

Parameters
  • {Object} conf Optional : Object with configuration properties.
  • {String} conf.content Optional : Sets content by: static content, DOM selector or URL. By default, the content is empty.
  • {Number || String} conf.width Optional : Sets width property of the component's layout. By default, the width is "500px".
  • {Number || String} conf.height Optional : Sets height property of the component's layout. By default, the height is elastic.
  • {Boolean} conf.fx Optional : Enable or disable UI effects. By default, the effects are enable.
  • {String} conf.event Optional : Sets the event ("click" or "hover") that trigger show method. By default, the event is "hover".
  • {String} conf.points Optional : Sets the points where component will be positioned, specified by configuration or centered by default: "cm cm".
  • {String} conf.offset Optional : Sets the offset in pixels that component will be displaced from original position determined by points. It's specified by configuration or zero by default: "0 0".
  • {Number} conf.showTime Optional : Sets a delay time to show component's contents. By default, the value is 400ms.
  • {Number} conf.hideTime Optional : Sets a delay time to hide component's contents. By default, the value is 400ms.
  • {Boolean} conf.cache Optional : Enable or disable the content cache. By default, the cache is enable.
Returns
itself
See
  • ch.Tooltip
  • ch.Modal
  • ch.Zoom

Examples

// Create a new contextual layer with configuration.
var me = $(".some-element").layer({
    "content": "Some content here!",
    "width": "200px",
    "height": 50,
    "event": "click",
    "showTime": 600,
    "hideTime": 200,
    "offset": "10 -10",
    "cache": false,
    "points": "lt rt"
});
// Create a simple contextual layer
var me = $(".some-element").layer("Some content.");
// Now 'me' is a reference to the layer instance controller.
// You can set a new content by using 'me' like this: 
me.content("http://content.com/new/content");

type <public> {string}

This public property defines the component type. All instances are saved into a 'map', grouped by its type. You can reach for any or all of the components from a specific type with 'ch.instances'.

uid <public> {number}

The 'uid' is the Chico's unique instance identifier. Every instance has a different 'uid' property. You can see its value by reading the 'uid' property on any public instance.

content <public>

Sets and gets component content. To get the defined content just use the method without arguments, like 'me.content()'. To define a new content pass an argument to it, like 'me.content("new content")'. Use a valid URL to get content using AJAX. Use a CSS selector to get content from a DOM Element. Or just use a String with HTML code.

Examples

// Get the defined content
me.content();
// Set static content
me.content("Some static content");
// Set DOM content
me.content("#hiddenContent");
// Set AJAX content
me.content("http://chico.com/some/content.html");
Parameters

{string} content

Static content, DOM selector or URL. If argument is empty then will return the content.
