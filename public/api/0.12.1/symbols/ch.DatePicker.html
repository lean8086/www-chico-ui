ch.DatePicker (Class)

Documentation updated on Wed Nov 14 2012, generated by JsDoc Toolkit.

Overview

ch.DatePicker extends ch.Controls. Datepicker lets you select dates.

Requires

  • ch.Calendar

Returns

itself

Use

Create a new datePicker.

var widget = $(".example").datePicker();

Create a new Date Picker with configuration.

var widget = $(".example").datePicker({
	 "format": "MM/DD/YYYY",
	 "selected": "2011/12/25",
	 "from": "2010/12/25",
	 "to": "2012/12/25",
	 "monthsNames": ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"],
	 "weekdays": ["Su", "Mo", "Tu", "We", "Thu", "Fr", "Sa"]
});

Configuration

The following options lets you create a customized ch.DatePicker:

  • conf: Object Object with configuration properties.
  • conf.format: String Sets the date format. By default is "DD/MM/YYYY".
  • conf.selected: String Sets a date that should be selected by default. By default is the date of today.
  • conf.from: String Set a maximum selectable date.
  • conf.to: String Set a minimum selectable date.
  • conf.points: String Points to be positioned. See Positioner component. By default is "ct cb".
  • conf.monthsNames: Array By default is ["Enero", ... , "Diciembre"].
  • conf.weekdays: Array By default is ["Dom", ... , "Sab"].
  • conf.closable: Boolean Defines if floated component will be closed when a date is selected or not. By default it's "true".
