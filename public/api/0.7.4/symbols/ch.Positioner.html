Class Detail

ch.Positioner

Positioner is a utility that resolve positioning problem for all UI-Objects.

Examples

// First example
ch.positioner({
	element: $("#element1"),
	context: $("#context1"),
	points: "lt rt"		//  Element left-top point = Context left-bottom point
});
 
// Second example
ch.positioner({
	element: $("#element2"),
	context: $("#context2"),
	points: "lt lb"		//  Element center-middle point = Context center-middle point
});
Parameters
  • {object} o : Object with configuration properties
Returns
